# 青萍 V1.0

这是青萍的第一个正式开源版本。

青萍是一款完全运行在本地的桌面效率助手。它以悬浮气泡作为入口，把今日任务、专注计时、时间回顾、每日感想和阶段计划放在一个轻量面板中。

## 主要功能

- 今日任务、标签、快速计时与完成记录
- 45 分钟倒计时、正计时和分段专注
- 青萍环与柱状涟漪时间回顾
- 每日感想和五级心情评分
- 主要任务、阶段目标、进度、日期与笔记
- 可自定义主题、背景、面板、气泡图片和透明度
- 计时结束青萍浮出动画、休息计时与三套可试听音效
- 本地数据保存和历史 CSV 导出

## 下载

- Windows：`QingPing-V1.0.0-Windows-x64-Setup.exe`
- macOS：`QingPing-V1.0.0-macOS-universal.dmg`
- Linux：`QingPing-V1.0.0-Linux-x86_64.AppImage`

当前安装包未配置商业代码签名。Windows 和 macOS 首次启动时可能显示系统安全提示。

所有个人任务、感想和计划只保存在本地，不会上传到远程服务器。
