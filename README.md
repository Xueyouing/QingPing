# 青萍

清新简约的桌面任务、专注与知识卡片助手。当前版本保持分界面产品结构：今日、专注、知识、历史彼此独立，避免把所有功能堆在同一个界面里。

## 运行

```bash
npm.cmd start
```

也可以直接双击 `index.html` 使用静态版本。

## 主要功能

- 今日：添加任务，选择或自定义标签，查看待办、已专注时长和完成数。
- 专注：支持倒计时、正计时；倒计时可自定义分钟数，并加入多个后续计时段，直到任务完成后归档。
- 知识：保留轻量茶歇知识卡片，可随机切换。
- 历史：完成任务后按日期分组，点击某一天进入详情，查看任务、标签、实际投入和分段记录。
- 设置：支持青萍绿、浅湖蓝、荷粉绿主题；主题会同步影响背景、主按钮、进度环、气泡和面板色彩。

## 任务输入

任务输入支持简短格式：

```text
写日报 25m #工作
```

也可以在输入框右侧直接选择标签；选择“自定义”后会显示自定义标签输入框。任务创建后仍可在列表里修改标签。

## 本地存储

Electron 主进程负责轻量本地文件存储：

- `qingping-state.json`：完整应用状态。
- `qingping-history.json`：历史任务记录。

两个文件位于 Electron `userData` 目录，可从托盘菜单“打开数据目录”进入。静态版本会退回浏览器 `localStorage`。
