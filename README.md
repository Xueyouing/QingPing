# 青萍桌面助手

青萍是一款清新、轻量的 Electron 桌面效率助手。它以悬浮气泡作为入口，提供今日任务、专注计时、回顾、感想和阶段计划等功能，适合希望把任务推进和日常记录放在手边的桌面用户。

## 功能概览

- 今日任务：添加待办、设置标签、快速进入计时，并查看今日完成记录。
- 专注计时：支持倒计时和正计时，倒计时默认 45 分钟，可编辑时间预设。
- 休息提醒：计时结束后显示青萍浮出动画，可配置音效和休息计时。
- 回顾：以青萍环和涟漪图查看当天时间分布。
- 感想：每日记录标题、心情评分和文字感想。
- 阶段计划：记录当前主要任务、阶段目标、进度和路线。
- 桌面气泡：支持尺寸、透明度、是否显示计时，以及内置或本地自定义图片。

## 运行

```bash
npm install
npm.cmd start
```

检查语法：

```bash
npm.cmd run check
```

## 本地数据

桌面版数据保存在 Electron `userData` 目录中：

- `qingping-state.json`：应用状态、任务、设置、感想和计划。
- `qingping-history.json`：历史任务记录备份。

托盘菜单中的“打开数据目录”可以直接进入该目录。仓库不会提交用户本地数据。
